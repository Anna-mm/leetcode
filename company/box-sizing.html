content-box: width 就是内容本身的宽度。border-box: width 是盒子的宽度，即内容本身的宽度+padding+border。

Background-clip 绘制区域，默认值是 border-box，也就是说到边框内，不包含边框=内容宽度+padding

content-box: // 红色 box 的 宽度: width+padding*2=110 // 蓝色 box 的 宽度: width+padding*2+border*2=120

border-box: // 红色 box 的 宽度: width-border*2=90 // 蓝色 box 的 宽度: 90+5*2=100
